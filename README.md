# Jellyfin Favorited Songs Playlist

Automatically creates and updates a **Favorited Songs** playlist in Jellyfin containing all the music tracks a user has favorited. Whenever a user favorites or unfavorites a song, the playlist is updated automatically, making it easy to play all your favorited tracks in one place.

---

## Installation

1. Add the plugin manifest to your Jellyfin:

   ```
   https://raw.githubusercontent.com/Saturn745/jellyfin-favorited-songs-playlist/refs/heads/master/manifest.json
   ```
2. Install the plugin through the Jellyfin dashboard.
3. After installation, any time a user favorites a song, their **Favorited Songs** playlist will update automatically.
